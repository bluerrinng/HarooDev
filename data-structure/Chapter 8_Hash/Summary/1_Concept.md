# 해시의 개념

- 정말 많은 데이터가 존재하기에 이런 데이터를 효율적으로 저장하거나 탐색하는 방법

- 해시 함수를 사용해서 변환한 값을 인덱스로 삼아 키와 값을 저장해서 빠른 데이터 탐색을 제공하는 자료구조

**어떻게 빠르게 탐색?**
- 보통은 인덱스를 통해서 탐색을 하지만 해시구조는 ``key``를 활용해 데이터 탐색을 빠르게 

### 해시의 특징
- 단 방향으로 동작, 즉 키를 통해 값을 찾을 수 있지만 값을 통해서 키를 찾을 수는 없다
- 순차 탐색할 필요 없이 해시 함수를 활용해서 특정 값이 있는 위치를 바로 찾을 수 있다
- 해시 테이블은 키와 대응한 값이 저장되어 있는 공간
  - 각 데이터를 ``bucket`` 버킷이라고 부른다


### 해시를 사용하는 분야
- 데이터를 저장하거나 검색
- 보안이 필요한 곳
- 비밀번호 관리
- 데이터베이스 인덱싱
- 블록체인 

# 해시 함수 
c++에서는 ``map``이라는 자료형이 제공되는데 이 자료형은 해시와ㄱ거의 동일하게 동작한다. 
    - ``(unordered_map)`` 을 말하는 것이다 왜냐면 해시는 자동으로 정렬을 해주는 기능은 없다.
